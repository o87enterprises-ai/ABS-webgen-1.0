# ⚡ DeepSite - Quick Start (30 seconds)

## The One Command

```bash
./deepsite-launch.sh
```

That's it! Everything else happens automatically.

## What You'll See

```
✅ DeepSite running:   http://localhost:3000
✅ Remote tunnel:      https://your-random-url.trycloudflare.com
✅ AI Model:           CodeLlama-7B (HuggingFace)
✅ Database:           SQLite (local)
```

## Copy & Paste URLs

- **For you**: `http://localhost:3000`
- **For clients**: `https://your-random-url.trycloudflare.com`

## That's It!

1. Open `http://localhost:3000` in your browser
2. Create a website with AI prompts
3. Share the remote URL with anyone
4. Edit and refine with unlimited generations
5. Everything saves locally automatically

## Stop Everything

Press `Ctrl+C` in your terminal.

## Get Help

```bash
./deepsite-manage.sh status
./deepsite-manage.sh logs
./deepsite-manage.sh help
```
